## A.31 The Single Helmholtz Resonator

### A. One Opening

The current transfer function of a single resonator:

**Equation A.31-1:**
```
H(s) = U_o(s)/U_1(s) = 1 / [1 + RG + s(LG + RC) + s^2 LC] = omega_01^2 / [(s - s_hat_1)(s - s_hat_1*)]
```

Where `s = sigma + j*omega` is the complex frequency variable.

### Damping Conditions

**1. Less than critical damping** (typical for vocal resonances):

Condition:
```
(1 + RG) / LC > (1/4) * (R/L + G/C)^2
```

**Equation A.31-2** (pole locations):
```
s_hat_1, s_hat_1* = -(1/2)(R/L + G/C) +/- j * sqrt[(1+RG)/LC - (1/4)(R/L + G/C)^2]
s_hat_1 = sigma_1 + j*omega_1
s_hat_1* = sigma_1 - j*omega_1
```

Vocal resonances generally have high Q = omega_0 / 2*sigma, implying RG << 1.

**Equation A.31-3** (resonance frequency and bandwidth):
```
F_1 = (1/2*pi) * omega_1 = 1 / (2*pi*sqrt(LC))
B_1 = (-1/pi) * sigma_1 = R/(2*pi*L) + G/(2*pi*C)
```

**Equation A.31-4** (in acoustic quantities, L = rho*l_e/A, C = V/rho*c^2):
```
F_1 = (c / 2*pi) * sqrt(A / l_e*V)
```

**Key insight:** *The resonance frequency is inversely proportional to the square root of the volume V, the effective length l_e of the neck, and directly proportional to the square root of the neck area A.*

**2. More than critical damping:**

Condition:
```
(1 + RG) / LC < (1/4) * (R/L + G/C)^2
```

**Equation A.31-5:**
```
s_hat_1, s_hat_1* = -(1/2)(R/L + G/C) +/- sqrt[(1/4)(R/L + G/C)^2 - (1+RG)/LC]
```

The roots are real and different. This overcritical damping probably applies to the first formant of a voiced constrictive.

### B. Two Openings

Adding a branch R_2 + sL_2 paralleling the capacitance introduces a pole a_1 and a zero a_2.

**Equation A.31-6:**
```
H(s) = [L_2(s - a_2)] / [(L_1 + L_2)(s - a_1)] * [(omega_1^2 + sigma_1^2) / ((s - s_hat_1)(s - s_hat_1*))]
```

**Equation A.31-7** (resonance frequency with two openings):
```
f_1 = (c / 2*pi) * sqrt[(1/V) * (A_1/l_1e + A_2/l_2e)]
```

Where l_2e is the length and A_2 the area of the second opening. The resonance frequency increases with an increase in the conductivity index A/l of either opening.

## A.32 The Double Helmholtz Resonator

Historical references: Raleigh (1877), Crandall (1927), Chiba and Kajiyama (1941), Essner (1947), Joos (1948).

### Transfer Function

**Equation A.32-1:**
```
H_s = [L_3 / (L_1 + L_2 + L_3)] * [(s - a_2) / (s - a_1)] * [(omega_1^2 + sigma_1^2)(omega_2^2 + sigma_2^2)] / [((s + sigma_1)^2 + omega_1^2)((s + sigma_2)^2 + omega_2^2)]
```

### Uncoupled Resonance Frequencies

**Equation A.32-3** (front resonator alone):
```
F_01 = 1 / (2*pi*sqrt(L_1*C_1))
```

**Equation A.32-4** (back cavity alone):
```
F_02 = 1 / (2*pi*sqrt(L_2*C_2))
```

**Equation A.32-5** (with glottis branch effect):
```
F_02 = [1 / (2*pi*sqrt(L_2*C_2))] * sqrt[(L_2 + L_3) / L_3]
```

### Coupled Resonance Frequencies

**Equation A.32-6:**
```
F_1^2 + F_2^2 = (1 / 4*pi^2) * (1/L_1*C_1 + 1/L_3*C_2 + 1/L_2*C_1)
F_1^2 * F_2^2 = (1 / 4*pi^2)^2 * (1 / L_1*C_1*L_2*C_2) = F_01^2 * F_02^2
```

### Key Ratios

**Equation A.32-7** (inductance ratio):
```
k_L = L_1/L_2 = l_1e*A_2 / l_2e*A_1
```

**Equation A.32-8** (capacitance ratio = volume ratio):
```
k_C = C_1/C_2 = V_1/V_2
```

**Equation A.32-9** (resonance frequencies from ratios):
```
F_1,2^2 = [1 / (4*pi^2*L_1*C_1)] * [(1/2)(1 + k_L + k_L*k_C) -/+ sqrt((1/4)(1 + k_L + k_L*k_C)^2 - k_L*k_C)]
```

### Formant-Resonator Dependency

**Equation A.32-10** (sensitivity coefficients):
```
A = -2*(dF_1/F_1)/(dC_1/C_1) = -2*(dF_2/F_2)/(dC_2/C_2) = (F_02^2 - F_1^2) / (F_2^2 - F_1^2)
B = -2*(dF_2/F_2)/(dC_1/C_1) = -2*(dF_1/F_1)/(dC_2/C_2) = (F_2^2 - F_02^2) / (F_2^2 - F_1^2)
C = -2*(dF_1/F_1)/(dL_2/L_2) = -2*(dF_2/F_2)/(dL_2/L_2) = (F_2^2 - F_01^2) / (F_2^2 - F_1^2)
D = -2*(dF_2/F_2)/(dL_1/L_1) = -2*(dF_1/F_1)/(dL_2/L_2) = (F_01^2 - F_1^2) / (F_2^2 - F_1^2)
```

**Equation A.32-11** (conservation rule):
```
A + B = C + D = 1
```

This means the sum of percentage increases in the two resonance frequencies is always one-half of the percentage decrease in any resonator element.

**Equation A.32-12, A.32-13** (special case F_01 = F_02):
```
A = D = 1/(1 + F_2/F_1)
B = C = 1/(1 + F_1/F_2)
A + B = B + D = 1
```

### Bandwidth Contributions

**Equation A.32-14:**
```
B_1 = (G_1/2*pi*C_1)*A + (G_2/2*pi*C_2)*B + (R_1/2*pi*L_1)*C + (R_2/2*pi*L_2)*D
B_2 = (G_1/2*pi*C_1)*B + (G_2/2*pi*C_2)*A + (R_1/2*pi*L_1)*D + (R_2/2*pi*L_2)*C
```

### Pressure Distribution

**Equation A.32-15** (at F_1):
```
P_1/P_1(f=F_1) = 1 - F_1^2/F_02^2
```

**Equation A.32-16** (at F_2):
```
P_1/P_1(f=F_2) = 1 - F_2^2/F_02^2
```

## A.33 The Single Tube as an Acoustic Resonator

### Transfer Function

**Equation A.33-1:**
```
U_o/U_a = H(s) = 1/cosh(Gamma(s)) = 1/cosh[a*l + s*l/c] = K / [Pi(n=1 to inf) (1 - s/s_n)(1 - s/s_n*)]
```

Where l = tube length, c = speed of sound, a = attenuation constant.

**Equation A.33-2** (attenuation constant):
```
a = a_R + a_G = (1/2c) * (R/L + G/C)
```

R, L, G, C are resistance, inductance, conductance, and capacitance per unit length.

### Resonance Frequencies (Quarter-Wavelength)

**Equation A.33-3:**
```
cosh(Gamma(s)) = 0
Gamma(s) = +/- j*(pi/2)*(2n-1)
```

**Equation A.33-4:**
```
omega_n = +/- (pi/2)*(2n-1)*c/l
sigma_n = -(a_R + a_G)*c
```

**Equation A.33-5:**
```
F_n = (2n-1)*c/4l
B_n = c*(a_R + a_G)/pi
```

**Equation A.33-6** (small losses):
```
K = 1/cosh(a*l) ≈ 1
```

### Ideal Neutral Vowel [schwa]

The ideal neutral vowel is defined as sound from an idealized vocal tract of constant cross-sectional area and effective length of approximately **17.6 cm** (or more precisely c/2000 cm where c is the velocity of sound).

**Equation A.33-7:**
```
F_n = (2n-1) * 500 c/s
```

i.e., **F_1 = 500, F_2 = 1500, F_3 = 2500 c/s**, etc.

### Factorized Pole Expansion

**Equation A.33-8:**
```
U_o/U_a = k_rg(s) / [Pi(n=1 to g) (1 - s/s_hat_n)(1 - s/s_hat_n*)]
```

The k_rg factor of the single tube can be used irrespective of particular resonator configuration - the total length is the most important parameter.

### Half-Wavelength Resonances (Both Ends Open or Closed)

**Equation A.33-10:**
```
Z * tanh(Gamma(s)) = 0
```

**Equation A.33-11:**
```
omega_n = (n-1)*pi*c/l_e
```

**Equation A.33-12:**
```
F_n = (n-1)*c/2*l_e
```

Where `l_e = l + l_t` (physical length plus end corrections).

### End Corrections

For open end with spherical baffle of radius 9 cm:
```
l_t = 0.8*(A/pi)^(1/2)  [Eq. 1.21-17]
```

Without baffle:
```
l_t = 0.6*(A/pi)^(1/2)  [Morse 1948]
```

**Equation A.33-13** (high impedance termination):
```
l_t = -c^2*A_g / (omega^2*A*l_g)
```

**Equation A.33-14** (with resistance):
```
l_t = -c*Z*L_g / (R_g^2 + omega^2*L_g^2)
```

**Equation A.33-15** (positive end correction, radiation reactance):
```
l_t = l_g*A/A_g
```

**Equation A.33-16** (capacitance termination):
```
C_g = l_g*A_g/rho*c^2
```

**Equation A.33-17:**
```
l_t = l_g*A_g/A
```

**Equation A.33-18** (large capacitance termination):
```
l_t = -c / (Z*omega^2*C_g) = -c^2*A / (omega^2*A_n*l)
```

### Damping Constants for Different Terminations

**Equation A.33-27:**

| Termination | Damping constant sigma |
|-------------|------------------------|
| A (small inductance) | -c[a_R(1 - 2*l_t/l_e) + a_G + a_t] |
| B (large inductance) | -c[a_R(1 - 2*l_t/l_e) + a_G + a_t(1 - 2*l_t/l_e)] |
| C (small capacitance) | -c[a_R + a_G(1 - 2*l_t/l_e) + a_t] |
| D (large capacitance) | -c[a_R + a_G(1 - 2*l_t/l_e) + a_t(1 - 2*l_t/l_e)] |

**Equation A.33-28** (approximate damping constant):
```
sigma = -c*(a_R + a_G + a_t)
```

**Equation A.33-29** (type D termination correction):
```
l_t_D = l_g * (A/A_g) * [(1/3) - c^2/(omega^2*l_g^2) + (0.48*A_g)/(l_g*A^(1/2)) * (1 - 1.25*(A_g/A)^(1/2))]
```

## A.34 Four-Tube Systems: Transform Equations for Arbitrary Source Locations

### Pole Function

**Equation A.34-1:**
```
U_o/U_q = H_p(s) = 1 / [cosh(Gamma_1)*cosh(Gamma_2)*cosh(Gamma_3)*cosh(Gamma_4)*(AB + CD)]
```

Where:
```
A = (1 + A_2/A_1 * tanh(Gamma_2)*tanh(Gamma_1))
B = (1 + A_4/A_3 * tanh(Gamma_4)*tanh(Gamma_3))
C = A_3/A_2 * (tanh(Gamma_2) + A_2/A_1 * tanh(Gamma_1))
D = A_4/A_3 * (tanh(Gamma_4) + tanh(Gamma_3))
```

**Equation A.34-2:**
```
H_p(s) = 1 / [Pi(n=1 to inf) (1 - s/s_hat_n)(1 - s/s_hat_n*)]
```

### Total Transfer Function

**Equation A.34-3:**
```
U_o(s)/E_s(s) = H(s) = H_p(s)*H_z(s)
```

Where H_p(s) is the pole function and H_z(s) is the zero function.

**Equation A.34-4:**
```
H_z(s) = [sinh(Gamma_1)*sinh(Gamma_2)*sinh(Gamma_3)*sinh(Gamma_4)*(ef + gh)] / Z_4
```

### Twin-Tube Pole Function

Setting Gamma_3 = Gamma_4 = 0:

**Equation A.34-8:**
```
H_p(s) = 1 / [cosh(Gamma_1)*cosh(Gamma_2) * (1 + A_2/A_1 * tanh(Gamma_1)*tanh(Gamma_2))]
```

### Resonance and Anti-Resonance Frequencies

**Equation A.34-20:**
```
F_n = omega_n/2*pi = (2n-1)*c/4*l_e
```

**Equation A.34-21** (anti-resonances):
```
F_m = omega_m/2*pi = n*c/2*l_2
```

### Pressure Response

**Equation A.34-22:**
```
P(omega, l_d) = (Delta_p / 4*pi*l_d) * K_T(omega) * H_z(omega) * H_p(omega)
```

### Empirical Damping Formula

**Equation A.34-23:**
```
a = 0.007*(pi/A)^(1/2) neper/cm
```

Assuming a_G = a_R = a/2. For an ideal hard-walled tube, a_G ≈ 0.45*a_R. The attenuation constants of vocal cavities are of the order of **2-8 times as large**.

## A.35 Damping Effect of Series and Shunt Losses Within Twin-Tube Resonators

### Pole Frequency Approximation

**Equation A.35-1:**
```
(A_2/A_1) * tan(omega*l_1/c) = cot(omega*l_2/c)
```

### Damping Constant Sum Rule

**Equation A.35-3:**
```
sigma = -c * [k_R1*a_R1 + k_G1*a_G1 + k_R2*a_R2 + k_G2*a_G2]
```

**Equation A.35-4:**
```
k_R1 + k_R2 = k_G1 + k_G2 = 1
```

The sum of the damping coefficients for series losses equals unity, as does the sum for parallel losses.

### Damping Coefficients

**Equation A.35-5a:**
```
k_R1/k_G1 = [1 + tan^2(phi_1) +/- tan(phi_1)/phi_1] / [1 + (A_1*l_2)/(A_2*l_1) + (1 + l_2*A_2/(l_1*A_1))*tan^2(phi_1)]
```

**Equation A.35-5b:**
```
k_R2/k_G2 = [1 + cot^2(phi_2) +/- cot(phi_2)/phi_2] / [1 + (A_2*l_1)/(A_1*l_2) + (1 + l_1*A_1/(l_2*A_2))*cot^2(phi_2)]
```

Where phi_1 = omega*l_1/c and phi_2 = omega*l_2/c.

### Special Cases

For ideal Helmholtz resonator: k_R1 = k_G2 = 1 and k_R2 = k_G1 = 0.

For standing wave resonance confined to tube 2: k_R1 = k_G1 = 0 and k_R2 = k_G2 = 1.

**Equation A.35-6** (Helmholtz approximation):
```
omega ≈ c*(A_1/A_2*l_1*l_2)^(1/2)
```

**Equation A.35-7:**
```
sigma ≈ -c * [(2*a_R1 + a_R2*l_2*A_1/l_1*A_2) / (2 + l_2*A_1/l_1*A_2) + a_G2]
```

**Equation A.35-8** (when l_2*A_1/l_1*A_2 << 1):
```
sigma = -c*(a_R1 + a_G2)
```

**Equation A.35-10:**
```
omega_n = pi*(n-1)*c/l_2e
```

## A.36 Summary of Twin-Tube Formulas for the Study of Resonator Damping

### Bandwidth Definition

**Equation A.36-1:**
```
B_n = -sigma_n/pi
```

### A. The Helmholtz Resonator

**Equation A.36-2:**
```
B = R/(2*pi*L_e) + G/(2*pi*C) = B_R + B_G
```

**Equation A.36-3** (resonance frequency):
```
F = (1/2*pi) * (1/L_e*C)^(1/2) = (c/2*pi) * (A/l_e*V)^(1/2)
```

### 1. Radiation Resistance

**Equation A.36-4:**
```
B_R0 = c*A^2*K(f) / (8^2*pi^2*V*l_e^2) = 45*(A/l_e)^2*(10/V)*K_s(f)
```

**Equation A.36-5:**
```
B_R0 = f^2*A*K_s(f) / (2*c*l_e) = 14*(f/1000)^2*(A/l_e)*K_s(f)
```

**Equation A.36-6:**
```
B_R0 = 2*pi*f^4*V*K_s(f) / c^3 = 4.5*(f/1000)^4*(V/10)*K_s(f)
```

### 2. Frictional Losses in Resonator Neck

**Equation A.36-7:**
```
B_Rf = S_A*(mu*f/rho*A)^(1/2) = 13*(f/1000*A)^(1/2)*S_A
```

**Equation A.36-8:**
```
B_Rf = (mu*c/2*pi*rho)^(1/2)*(1/l_e*A*V)^(1/2)*S_A = 9.7*(100/A*l_e*V)^(1/2)*S_A
```

**Equation A.36-9:**
```
B_Rf = (c/2*pi)*(mu/rho*l_e*V*f)^(1/2)*(100/V*l_e)^(1/2)*S_A = 7.3*(1000/f)^(1/2)*(100/V*l_e)^(1/2)*S_A
```

### 3. Turbulent Air Stream Effect

**Equation A.36-11:**
```
B_D1 = R_D / (2*pi*L) = v/2*pi*l
```

The corresponding time constant 1/pi*B_D1 is of magnitude equal to the time for a particle in the superimposed DC flow to propagate down the tube.

### Bandwidths from Glottis Damping

**Equation A.36-12:**
```
B_g1 = 1/(2*pi*R_g*C_s) = c*A_g/(2*pi*v_g*V)
```

**Equation A.36-13:**
```
B_t1 = 1/(2*pi*R_t*C_s) = A_t*c/(2*pi*V_t^2)
```

**Equation A.36-14:**
```
B_t2 = Z_s*c/(pi*R_t*V_t) = (rho*c^2)/(pi*R_t*V_t) = A_t*c/(pi*V_t)
```

### 4. Heat Conduction and Cavity Wall Vibration

**Equation A.36-15:**
```
B_G = 0.45*(A_1/A_2)^(1/2)*B_R
```

**Equation A.36-16:**
```
B_G = S(x-1)/A * [K_h*f*c / (4*pi*rho*C_p)]^(1/2) = 5.8*(f/1000*A_2)^(1/2)*S_A2
```

**Equation A.36-17:**
```
B_Gw = G'_w / (2*pi*C'_2)
```

Van den Berg (1953) estimates wall vibration bandwidth contribution of **25-50 c/s at 300 c/s**, proportional to f^(-2.5).

### B. Standing Waves in Tubes

### 1. Radiation Damping of an Open Tube

**Equation A.36-18:**
```
B_R0 = R_0*c / (Z*l_e*pi) = f^2*A / (c*l_e) * K_s(f)
```

Condition: R_0/Z < 0.5

**Equation A.36-19** (normalized to l_e = 17.6 cm, A = 8 cm^2):
```
B_R0 = 29 * (f/1500)^2 * (A/8) * (17.6/l_e) * K_s(f)
```

At 1500 c/s (second resonance), K_s(f) = 1.6 and B_R0 = 46 c/s.
At 500 c/s (fundamental), B_R0 = 3.9 c/s.
B_3,R0 = 137 c/s, B_4,R0 = 224 c/s.

### 2. Radiation Damping in Tube with Narrow Opening

**Equation A.36-20:**
```
B_R0 = c*A_1^2 / (4*pi^2*l_1e^2*l_2*A_2) * K_s(f)
```

**Equation A.36-21:**
```
B_R0 = 90*(A_1/l_1e)^2*(10/l_2*A_2) * K_s(f)
```

The bandwidth decreases with the square of the lip-rounding parameter l_1e/A_1. **Rounded vowels have narrower formant bandwidths than unrounded vowels.**

### 3. Cavity Wall Damping of Standing Waves

**Equation A.36-22:**
```
B_a = (c/pi)*(a_R + a_G) = R/(2*pi*L) + G/(2*pi*C) = B_R + B_G
```

**Equation A.36-23:**
```
B_a = 18.5 * (f/1000 * 1/A)^(1/2) * S_A
B_G = 0.31 * B_a
B_R = 0.69 * B_a
```

## Summary for Klatt Synthesizer Implementation

### Key Formulas for Formant Synthesis

1. **Resonance frequency** depends on sqrt(A/l_e*V) for Helmholtz resonators

2. **Bandwidth** has multiple components:
   - Radiation resistance (increases with f^2)
   - Frictional losses (increases with f^(1/2))
   - Wall vibration (decreases with f^(-2.5))
   - Glottal damping

3. **Lip rounding effect**: Rounded vowels have narrower bandwidths because B_R0 decreases with (l_1e/A_1)^2

4. **Formant coupling**: For double resonators, F_1 * F_2 = F_01 * F_02 (product of coupled frequencies equals product of uncoupled frequencies)

5. **Damping coefficient sum rule**: k_R1 + k_R2 = k_G1 + k_G2 = 1

### Values for Synthesis

- Neutral vowel: F1=500, F2=1500, F3=2500, F4=3500 Hz
- Effective vocal tract length: ~17.6 cm
- Higher formant bandwidths increase significantly with frequency (radiation damping dominates)
