This paper presents a complete glottal excitation model combining the LF parametric waveform with pitch-synchronous amplitude-modulated Gaussian noise for singing voice synthesis. The key contribution is the empirical parameter table relating the single Rd parameter to voice quality (pressed/normal/breathy) along with analysis algorithms using wavelet packet decomposition to separate periodic and aperiodic components. This work provides actionable parameter values and noise modeling details directly applicable to improving voice quality control in the Qlatt synthesizer's LF source implementation.
